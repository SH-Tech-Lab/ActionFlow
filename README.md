# ActionFlow

ActionFlow is a lightweight and user-friendly Android task management application built with React Native and Expo. It helps users organize daily tasks with a simple, clean interface while providing a smooth mobile experience.

This project is developed and maintained by **SH Tech Lab** as part of our mobile application portfolio.

---

##  Features

* Create new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* View all tasks
* Local data storage
* Clean and responsive user interface
* Fast and lightweight performance

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/SH-Tech-Lab/ActionFlow.git
cd ActionFlow
```

### Install dependencies

```bash
pnpm install
```

### Start the development server

```bash
pnpm start
```
